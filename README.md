# AI Trip Planner 🌍

An AI-powered travel planner built with Django and Google Gemini AI.

## Features

- User Registration & Login
- AI Trip Itinerary Generation
- Dashboard
- Search Trips
- Edit & Delete Trips
- Favorite Trips
- Live Weather
- Google Maps
- AI Packing Checklist
- PDF Download

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite
- Google Gemini AI
- OpenWeather API

## Installation

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
